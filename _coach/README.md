# Coach Materials

This folder contains materials for the coach running the course. The student doesn't need to look in here — but if they do, nothing here is secret.

## Contents

| File / Folder | What it is |
|---------------|------------|
| `CLAUDE.md` | Full project context for Claude Code — use this in your own sessions when planning or reviewing |
| `session-guide.md` | How to run sessions: structure, timing, what to watch for month by month |
| `curriculum/` | Full month-by-month curriculum including facilitator notes (the student versions in `../curriculum/` have these stripped) |

## How the coach folder relates to the rest of the repo

The student-facing curriculum in `../curriculum/month-X.md` contains everything the student needs for each week: the opening question, legal content, AI skill, activities, and journal prompt. Coach notes (end-of-month checks, session tips, things to watch for) are kept here so the student's view is clean.

If you're setting this up as a template for a new student, the student can safely clone the whole repo — the `_coach/` prefix is a signal, but nothing here will spoil the learning experience if they read it.
