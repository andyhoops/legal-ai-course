# Tools and Templates

This folder contains the reusable tools and templates I've built over the course — things that make my research workflow faster or more consistent.

## What goes here

- **Case analysis templates** — structured formats for analysing a new case
- **Research workflow checklists** — step-by-step process for tackling a legal question
- **CLAUDE.md snippets** — reusable context blocks I've found useful
- **Anything else** that a future version of me (or someone else) could pick up and use

## Goal

By the end of the course: at least one tool or template that a stranger could pick up and use without explanation.
