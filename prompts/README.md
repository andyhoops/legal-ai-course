# My Prompt Library

This folder contains the reusable prompts I've developed over the course — the ones worth keeping and using again.

## How to add a prompt

Create a new Markdown file for each prompt (or group of related prompts). For each entry include:

- **Name** — what you call it
- **The prompt** — the actual text, ready to copy and paste
- **When to use it** — what task it's designed for
- **Example output** — paste a snippet of a good result you got with it
- **Notes** — what to adjust for different situations

## Goal

By the end of the course: 10+ prompts covering the main legal research tasks you've practiced.
