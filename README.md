# My Legal AI Learning Journey

*A 4-month exploration of UK contract law using Claude Code as a learning partner.*

---

This is my personal learning journal. I'm using AI — specifically Claude Code — to explore UK contract law, develop research and analysis skills, and document my thinking as I go.

By the end of the course I'll have:
- Researched landmark UK contract law cases and built genuine legal knowledge
- Built a library of reusable prompts for legal analysis
- Developed workflows for AI-assisted research
- Written 16 weeks of honest reflections on what it means to learn with AI

## What's in this repo

| Folder / File | Contents |
|---------------|----------|
| `getting-started.md` | Start here |

## The course at a glance

4 months, roughly one session per week:

| Month | Theme |
|-------|-------|
| 1 | Foundations — AI basics, contract law intro, first prompts |
| 2 | Going Deeper — Context, critical evaluation, comparing cases |
| 3 | Building Tools — Prompt library, templates, reusable workflows |
| 4 | Portfolio — GitHub Pages, capstone project, sign-off |

See [`curriculum/overview.md`](curriculum/overview.md) for the full week-by-week plan.
